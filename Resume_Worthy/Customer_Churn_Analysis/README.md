# Enterprise Customer Churn & Subscriber Attrition Analytics (Portfolio Asset)

An industry-grade analytics platform tracking customer retention vectors, missing value handling paradigms, behavioral correlations, and operational account inefficiencies using Seaborn.

## Core Architectural Layers
* **Data Hygiene Handling:** Deploys defensive two-step datatype casting on raw data values (`Support Calls`, `Total Spend`) to protect pipeline arrays from text reduction runtime limits.
* **Feature Relationship Mapping:** Tracks engagement dimensions (`Usage Frequency`), support ticket frequencies, and subscription categories against structural account drops.
* **Multi-Asset Delivery Suite:** Automatically isolates and exports 4 standalone high-resolution charts, 1 combined executive presentation dashboard layout, and comprehensive correlation space matrix maps.

## Strategic Operational Recommendations
1. **Friction Threshold Interventions:** Accounts reaching more than 4 support entries present high churn risk. Customer success managers should implement automated outreach systems for this trigger point.
2. **Contractual Adjustments:** Rolling monthly terms show severe customer leakage over long-term annual accounts. Marketing squads should configure upfront credit incentives to transition high-value monthly accounts onto long-term structures.

## Execution Guide
1. Place the dataset at `data/customer_churn.csv`.
2. Install framework dependencies: `pip install pandas numpy matplotlib seaborn`
3. Execute data science pipeline engine: `python main.py`
