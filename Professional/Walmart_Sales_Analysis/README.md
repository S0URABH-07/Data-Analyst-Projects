# Enterprise Walmart Store Sales & Market Elasticity Analysis (Advanced Portfolio Piece)

An advanced corporate business intelligence pipeline analyzing historical store sales performance, macroeconomic conditions, and seasonal demand amplification using Seaborn.

## Core Engineered Architecture
* **Financial Margin Visualizer:** Computes and monitors calculated revenue transformations and estimated profit distributions.
* **Macro Market Segmentations:** Structures dynamic classification tiers based on total transaction density rather than raw location metrics.
* **Multi-Asset Delivery Suite:** Automatically isolates and extracts 3 individual statistical graphics, 1 unified high-resolution 4-panel courtroom layout presentation dashboard, and multi-variable covariance data maps.

## How to Run:
1. Download the source dataset file and place it in `data/walmart_sales.csv`.
2. Install explicit package dependencies: `pip install pandas numpy matplotlib seaborn`
3. Execute engine: `python main.py`
