# Enterprise Airbnb Market Intelligence & Asset Yield Analysis (Advanced Portfolio)

An advanced, industry-grade property-technology (PropTech) optimization and predictive feature preparation engine evaluating hospitality supply networks, spatial distribution configurations, and annual revenue capacities using Seaborn.

## Core Engineered Architecture
* **Financial Performance Indicators:** Generates custom optimization markers checking Estimated Annual Revenue Capacity (`Est_Annual_Revenue`) and Booking Occupancy proxies.
* **Pricing Layer Stratification:** Segments the market using quantile cuts over continuous listing numbers (`price`) into explicit actionable asset types.
* **Multi-Asset Delivery Suite:** Automatically isolates and saves 4 standalone data science graphics, 1 high-resolution 4-panel courtroom layout presentation dashboard, and predictive feature covariance maps.

## How to Run:
1. Download the source dataset file and place it inside your workspace directory at `data/airbnb_listings.csv`.
2. Install standard dependencies: `pip install pandas numpy matplotlib seaborn`
3. Execute engine: `python main.py`
