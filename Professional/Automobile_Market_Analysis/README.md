# Enterprise Automobile Market Analytics & Valuation Engine (Advanced Portfolio)

An advanced, corporate business intelligence pipeline evaluating vehicle asset depreciation risks, multi-variable engine performance output scales, and manufacturer pricing resilience using Seaborn.

## Core Engineered Architecture
* **Valitation Elasticity Tracking:** Integrates dynamic calculations measuring performance configurations against real consumer price trends.
* **Feature Covariance Matrices:** Compiles multi-variable interaction spaces (Horsepower, MPG limits, Mileage drops) to isolate depreciation indicators.
* **Multi-Asset Delivery Suite:** Automatically isolates and saves 4 standalone data science graphics, 1 high-resolution 4-panel courtroom layout presentation dashboard, and an executive PDF briefing.

## How to Run:
1. Initialize the required directories and data files: `python main.py`
2. Install standard framework dependencies: `pip install pandas numpy matplotlib seaborn weasyprint`
