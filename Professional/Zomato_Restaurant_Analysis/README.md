# Enterprise Zomato Store Sales & Customer Experience Optimization (Advanced Portfolio)

An advanced, industry-grade food-tech analytics pipeline evaluating restaurant pricing tiers, delivery system integrations, and geographical market density metrics using Seaborn.

## Core Engineered Architecture
* **Standardized Pricing Analysis:** Normalizes transactional metrics specifically to base currency standards (INR) to conduct clear consumer pricing reviews.
* **Customer Engagement Indices:** Structures custom composite indexes (`Aggregate rating` multiplied by `Votes`) to bypass reviewer selection bias and find true local demand hubs.
* **Multi-Asset Delivery Suite:** Automatically isolates and exports 4 standalone statistical graphics, 1 unified high-resolution 4-panel master presentation dashboard, and comprehensive feature correlation maps.

## How to Run:
1. Download the source dataset and place it inside `data/zomato.csv`.
2. Install dependencies: `pip install pandas numpy matplotlib seaborn`
3. Execute engine: `python main.py`
