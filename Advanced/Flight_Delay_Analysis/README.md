# Commercial Aviation Flight Delay & Logistics Resilience (Advanced Portfolio Piece)

An advanced, industry-grade transportation operations and aviation logistical pipeline tracking system utilizing Seaborn to unpack network disruptions.

## Strategic Architectural Components
* **Logistical Disruption Performance Metrics:** Engineers dynamic fields evaluating Systemic Delay Rates, Disruption Intensities, and Cancellation Probabilities.
* **Network Saturated Densities (EDA):** Charts multi-dimensional traffic load scales against system latency trends.
* **Multi-Asset Delivery Pipeline:** Automatically isolates and exports 3 core standalone analytical graphics, 1 master combined executive dashboard, and comprehensive correlation maps.

## How to Run:
1. Place `Airline_Delay_Cause.csv` in your `data/` folder directory.
2. Install dependencies: `pip install pandas numpy matplotlib seaborn`
3. Execute engine: `python main.py`
