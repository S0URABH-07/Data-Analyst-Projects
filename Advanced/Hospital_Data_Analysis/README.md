# Enterprise Hospital Data Systems Analysis (Advanced Portfolio Piece)

An advanced, industry-grade clinical operations and hospital management systems optimization dashboard using Seaborn to analyze patient flow dynamics.

## Strategic Architectural Components
* **Operational Performance Metrics:** Engineers dynamic fields for patient Length of Stay (LOS) and Cost Per Day vectors.
* **Epidemiology Market Shares:** Tracks disease distribution tracking maps and payer coverage weights.
* **Multi-Asset Delivery Pipeline:** Automatically isolates and exports 3 core analytical graphics, 1 master multi-axis dashboard, and comprehensive correlation maps.

## How to Run:
1. Place `healthcare_dataset.csv` in your `data/` folder and name it `hospital_patients.csv`.
2. Install dependencies: `pip install pandas numpy matplotlib seaborn`
3. Execute engine: `python main.py`
