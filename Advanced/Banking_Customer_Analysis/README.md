# Enterprise Banking Customer Credit Analysis (Advanced Portfolio)

An advanced industry-grade banking underwriting and credit distribution engine utilizing Seaborn to model financial demographic feature behaviors.

## Core Engineered Architecture
* **Risk Density Quantification (KDE):** Evaluates multi-dimensional continuous income and age curves natively.
* **Feature Engineering:** Compiles Debt-to-Income and Asset wealth multipliers proxy frameworks.
* **Multi-Asset Delivery Suite:** Automatically isolates and extracts 4 individual plots, 1 high-resolution combined executive dashboard, and a complete multi-dimensional pairwise distribution matrix grid.

## How to Run:
1. Install requirements: `pip install pandas numpy matplotlib seaborn`
2. Run code: `python main.py`
